Missing values

The modality for representing the Stata missing values can be selected by the user.

For example, choose "letters" for representing the Stata missing values ".", ".a", ".b", ..., ".z" as ".", "a", "b", ..., "z" respectively. A string like "-", ".", "m", "NA" and "NaN" can also be chosen to represent all the Stata missing values.

If "IEEE 754" is chosen, the Stata missing values will be represented according to their inner IEEE 754 numeric representation. For example "." will be represented as "8.9884656743115795E+307" and ".a" as "8.990660123939097E+307".

Decimals places for columns

Type here a comma-separated list of numbers where each number represents the number of digits to the right of the decimal separator to be used for each column of the inserted matrix. The first number in the list refers to the first column, the second refers to the second column and so on. For example, type "3, 5, 2" if 3 decimal places for the first column, 5 decimal places for the second column and 2 decimal places for the third column are required.

If the number of columns is greater than the list length, then the last number on the list will apply to all the columns for which nothing has been specified. For example typing "3, 5" for a matrix with 4 columns is equivalent to typing "3, 5, 5, 5". If only one number is inputted, that number will be applied to all the columns.

A number of decimal places between 0 and 20 can be set for each column.
